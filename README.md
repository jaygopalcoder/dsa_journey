# DSA Journey: From Logic to Optimization

This repository is my personal laboratory for mastering Data Structures and Algorithms. I am not just solving problems; I am building the foundational mental models required for high-level AI/ML engineering.

## 🛠️ The Stack
- **Language:** Python 3.12+
- **Environment Management:** [uv](https://github.com/astral-sh/uv) (Fast, Rust-based package manager)
- **Primary Resources:** *Grokking Algorithms* (Bhargava), *Algorithms in a Nutshell* (O'Reilly)

## 🎯 My Philosophy
1. **Grok the Concept:** Understand the visual and mathematical logic first.
2. **Implement from Scratch:** Write the algorithm in pure Python (no libraries) to master the "how."
3. **Optimize & Test:** Solve related LeetCode problems and analyze Time/Space complexity ($O(n)$).
4. **Repeat:** Consistent daily deep-work blocks.
